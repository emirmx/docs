## Introduction

In addition to the integration capabilities available in the Studio Pro, Mendix also supports integration with external systems through the RapidMiner Platform. Using Altair AI Studio and AI Hub, developers can quickly build enterprise-grade applications that seamlessly integrate with RapidMiner Platform's powerful machine learning model capabilities. Graph Studio further provides a scalable environment for connecting, managing, and analyzing graph-based data.
