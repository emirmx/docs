## Introduction

The Siemens UI Module provides the foundational user interface implementation for Siemens‑aligned Mendix apps. It delivers a consolidated set of layouts, styling, and core UI components that enable development teams to build consistent user experiences in line with the Siemens Design Language.

The module is based on the DI SW Product Design System, and includes common UI foundations that can be reused across Mendix apps. To support different design system entry points used across Siemens environments, the module also includes selected structural elements from the iX Design System, such as the application frame and icon set.

This enables teams to:

* Use an iX‑based page layout when required for alignment with existing applications.
* Use a DI SW‑based page layout when building directly on the DI SW Product Design System.

All UI components follow a single, consistent implementation approach.

## Intended Use Cases

The Siemens UI Module is intended as a base UI layer for Mendix apps developed by:

* Siemens customers
* Siemens solution partners
* Development teams building Siemens‑aligned solutions on Mendix

It provides a stable foundation on top of which application‑specific UI, page templates, and components can be built.
