## Introduction 

The Mendix platform enables developers to build powerful agentic systems by using the Model Context Protocol (MCP) to expose and consume logic from external systems. The modules help to facilitate a client-server connection to consume tools and prompts ([MCP Client module](/appstore/modules/genai/model-context-protocol/mcp-client/)) or to expose Mendix logic, such as microflows, to external AI systems ([MCP Server module](/appstore/modules/genai/model-context-protocol/mcp-server/)).
