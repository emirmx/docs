These release notes cover changes made to the [Private Mendix Platform](/private-mendix-platform/) in version 1.24.

{{% alert color="info" %}}
This is the [LTS](/releasenotes/studio-pro/lts-mts/#lts) version release. Customers are encouraged to upgrade to this version. [Previous MTS versions](/releasenotes/private-platform/1-13/) of Private Mendix Platform are no longer supported.
{{% /alert %}}

## 1.24.0

### Release date: September 12, 2025

Version 1.24.0 of Private Mendix Platform introduces a number of updates and fixes.

#### Updates

##### Studio Pro Updates

* Studio Pro 9.24 LTS latest patch version updated to [9.24.37](/releasenotes/studio-pro/9.24/#92437)
* Studio Pro 10.24 LTS latest patch version updated to [10.24.5](/releasenotes/studio-pro/10.24.5/#10245)
* Studio Pro 10.6 and 10.12 are deprecated as of this release, and customers are encouraged to upgrade to Studio Pro 10.24 LTS

##### Other Platform Components

* Mendix on Kubernetes Operator version [2.23.0](/releasenotes/developer-portal/mendix-for-private-cloud/#2.23.0)
* Private Cloud License Manager (PCLM) version [0.10.4](/releasenotes/developer-portal/mendix-for-private-cloud/#license-manage-cli-v0104)

#### Fixes

We have provided the following fixes:

* We have fixed vulnerabilities related to the following:

    * CVE-2025-49146
    * CVE-2025-24970

* (UI) We have made other, minor UI improvements and fixes.
